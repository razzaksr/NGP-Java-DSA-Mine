**A segment tree is a binary tree data structure that allows efficient range queries (like sum, minimum, maximum) and updates in \(O(\log n)\) time.** Itâ€™s widely used in competitive programming and scenarios where you need to query and update subarrays quickly.

## ðŸŒ³ What is a Segment Tree?
- A **segment tree** stores information about intervals (segments) of an array.
- Each node represents a segment \([l, r]\) and stores a value (like sum, min, or max).
- The root represents the entire array, and leaves represent single elements.

---

## âš¡ Operations Supported
- **Range Queries**:
  - **Range Sum Query (RSQ):** Find the sum of elements in a given range.
  - **Range Minimum Query (RMQ):** Find the minimum element in a given range.
  - **Range Maximum Query (RMaxQ):** Find the maximum element in a given range.
- **Updates**:
  - Point updates (change a single element).
  - Range updates (add/assign values to a subarray).

Both queries and updates run in **\(O(\log n)\)** time, compared to \(O(n)\) for naive solutions.
